# Source - Codigo Fonte 
 
## Organizacao 
 
- **Shared/** - Classes compartilhadas entre Service e Client 
- **DAO/** - Data Access Objects (acesso ao banco) 
- **ACBr/** - Integracao com componentes ACBr 
- **Service/** - Aplicacao de servico Windows 
- **Client/** - Aplicacao desktop com interface 
 
## Convencoes 
 
- Units: prefixo `u` - ex: `uEmpresa.pas` 
- Forms: prefixo `ufrm` - ex: `ufrmMain.pas` 
- DAO: sufixo `DAO` - ex: `uEmpresaDAO.pas` 
- Classes: prefixo `T` - ex: `TEmpresa`
